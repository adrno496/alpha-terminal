🔑 Comment obtenir une clé API IA et données financières

Guide pratique 2026 — toutes les clés supportées par Alpha (14 LLMs + 11 providers de données). Liens directs, étapes, tier gratuit signalé.

📌 TL;DR — 7 providers IA gratuits permettent d'utiliser Alpha sans débourser un centime (GitHub Models, Cerebras, Mistral, NVIDIA, Cloudflare, HuggingFace, Cohere). Une seule clé suffit pour démarrer. Pour des analyses pro (10-K, DCF, fiscal complexe), passe sur Claude / GPT-4o / Gemini Pro (~$0,005-0,10 par analyse).

🚀 Démarrer en 30 secondes

  1. Choisis un provider gratuit ci-dessous (recommandé : GitHub Models si tu as déjà un compte GitHub).
  2. Clique sur le lien direct, crée un compte ou génère un token.
  3. Copie la clé.
  4. Ouvre Alpha → Paramètres → Clés API → colle la clé. Elle est chiffrée localement (AES-GCM 256).
  5. Lance ta première analyse 🎉

🎁 Providers IA gratuits (7)

Ces providers offrent un tier gratuit suffisant pour l'usage occasionnel à intensif. Idéal pour démarrer sans engagement.

🐙 GitHub Models 100% gratuit

Le plus simple si tu as déjà un compte GitHub. Rate-limit raisonnable. Modèles disponibles : GPT-4o, Phi-4, Llama 3, DeepSeek, Mistral.

https://github.com/settings/tokens
  1. Connecte-toi à ton compte GitHub.
  2. Va sur Settings → Developer settings → Personal access tokens → Fine-grained tokens (ou Classic).
  3. Clique Generate new token. Donne-lui un nom (ex: "Alpha Terminal").
  4. Aucun scope spécial requis pour GitHub Models. Crée-le et copie le token (commence par ghp_ ou github_pat_).

⚡ Cerebras tier gratuit

Inférence ultra-rapide (Llama 3.3 70B à >2000 tok/s). Free tier généreux : 1M tokens/jour.

https://cloud.cerebras.ai/?tab=api-keys
  1. Va sur cloud.cerebras.ai, inscris-toi avec ton email.
  2. Onglet API Keys, clique Create API Key.
  3. Copie la clé (commence par csk-).

🇫🇷 Mistral AI tier expérimentation

Modèles européens. Tier expérimentation gratuit pour Mistral Small/Medium. Codestral (code) inclus.

https://console.mistral.ai/api-keys/
  1. Va sur console.mistral.ai, inscris-toi.
  2. Active le plan Experiment (gratuit, rate-limit doux).
  3. Onglet API Keys → Create new key.

🟢 NVIDIA NIM 1000 crédits offerts

Catalogue de modèles open-source hostés par NVIDIA (Llama, Nemotron, DeepSeek). 1000 crédits free à l'inscription.

https://build.nvidia.com/explore/discover
  1. Inscris-toi sur build.nvidia.com avec ton email.
  2. Choisis n'importe quel modèle, clique Get API Key.
  3. Copie la clé (commence par nvapi-).

☁️ Cloudflare Workers AI 100K req/jour

Modèles open-source hostés sur l'edge Cloudflare. 100 000 requêtes/jour gratuites.

https://dash.cloudflare.com/profile/api-tokens
  1. Crée un compte Cloudflare (gratuit).
  2. Note ton Account ID dans le dashboard (sidebar droite).
  3. Va sur My Profile → API Tokens → Create Token. Choisis le template Workers AI.
  4. Alpha demande à la fois le token ET l'Account ID.

🤗 HuggingFace gratuit (rate-limited)

Inference Providers : accès à des centaines de modèles open-source via une seule clé.

https://huggingface.co/settings/tokens
  1. Inscris-toi sur huggingface.co.
  2. Settings → Access Tokens → New Token. Type : Read.
  3. Copie le token (commence par hf_).

🟦 Cohere clé d'essai

Bon en classification, embeddings et RAG multilingue. Clé d'essai gratuite à l'inscription.

https://dashboard.cohere.com/api-keys
  1. Inscris-toi sur dashboard.cohere.com.
  2. Onglet API Keys → Trial Key (la clé créée par défaut).
  3. Copie-la.

🔒 Sécurité des clés dans Alpha

  • Chiffrement local AES-GCM 256 bits avec dérivation PBKDF2 (100 000 itérations) à partir d'un mot de passe maître que tu choisis.
  • Aucun serveur Alpha — l'app appelle directement les APIs des providers depuis ton navigateur. Vérifiable via DevTools → Network.
  • Stockage IndexedDB local — tes clés ne quittent jamais ton appareil.
  • Backup chiffré exportable (.atb) — pour migrer ou sauvegarder.

❓ FAQ

J'ai une seule clé, est-ce que tous les modules fonctionnent ?

Oui. Le smart router d'Alpha sélectionne automatiquement le meilleur provider disponible pour chaque module. Avec une seule clé, les 46 modules tournent. Plus tu en ajoutes, plus l'app peut router intelligemment (ex: Grok pour le sentiment X, Claude pour les 10-K, Gemini pour les longs PDFs).

Combien ça coûte par mois ?

Avec providers gratuits seulement : 0€. Avec un mix gratuit + Claude/GPT pour les analyses critiques : $2-15/mois. Power user (3+ analyses jour, 10-K, research-agent) : $15-50/mois. Détails sur la page Coûts API.

Que faire si j'oublie mon mot de passe maître ?

Pas de récupération possible (c'est le prix de la sécurité 100% locale). Tu peux réinitialiser le vault → tu perds les clés stockées et il faut les recoller. Astuce : exporte un backup chiffré régulièrement et garde le mot de passe en lieu sûr (gestionnaire de mots de passe).

Les clés API sont-elles transmises à Alpha ?

Non. Jamais. L'app fonctionne en client-side : tes clés vivent uniquement dans IndexedDB de ton navigateur, et les requêtes IA partent directement de ton navigateur vers le provider. Tu peux mettre l'app offline (mode avion) après chargement et les analyses continueront à passer normalement.

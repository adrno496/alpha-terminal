🔒 Vérifiable techniquement

Vérifie toi-même qu'Alpha n'envoie aucune donnée à nos serveurs

On répète partout que tes analyses restent 100% locales. C'est facile à dire. Voici comment vérifier toi-même en 30 secondes avec les outils déjà installés dans ton navigateur. Aucune confiance aveugle requise.

Le tutoriel reproductible (Chrome / Firefox / Safari / Edge)

1

Ouvre Alpha Terminal

Rends-toi sur alpha-terminal.app et clique Démarrer. Tu peux entrer une clé API gratuite (GitHub Models, Cerebras…) ou cliquer "Accéder sans clé" pour explorer.

2

Ouvre les DevTools de ton navigateur

Raccourci clavier :

  • Mac : Cmd + Option + I (Chrome/Edge/Firefox) ou Cmd + Option + C (Safari, après avoir activé "Menu Développement" dans les Préférences)
  • Windows / Linux : F12 ou Ctrl + Shift + I

Va dans l'onglet Network (ou "Réseau"). Coche "Preserve log" en haut.

3

Filtre les requêtes par domaine

Dans la barre de filtre du panneau Network, tape :

alpha-terminal.app

Cela ne montrera que les requêtes vers nos serveurs (et pas vers Anthropic, OpenAI, Lemonsqueezy, etc.).

4

Lance une analyse

Choisis une action (ex: AAPL) et lance une Quick Analysis ou un 10-K Decoder. Tu verras défiler les requêtes en temps réel.

5

Observe le résultat

Pendant toute l'analyse, le panneau filtré sur alpha-terminal.app doit afficher :

📭 Aucune requête.

Toutes les requêtes partent directement vers le LLM provider que tu as choisi (api.anthropic.com, api.openai.com, generativelanguage.googleapis.com, etc.) — jamais vers nos serveurs.

Architecture technique : pourquoi c'est possible

Alpha n'a aucun backend pour les analyses. C'est un choix d'architecture délibéré. Voici comment chaque composant fonctionne :

🧠 LLMs Tes prompts vont directement à api.anthropic.com, api.openai.com, etc. via fetch() dans ton navigateur. Aucune route intermédiaire chez nous.
💾 Données perso Wealth, RAG, analyses passées : tout est stocké dans IndexedDB + localStorage du navigateur. Pas de DB serveur.
🔐 Vault clés API Tes clés sont chiffrées AES-GCM-256 avec ta passphrase (PBKDF2 100k iter). Le vault chiffré reste en local. On ne le voit jamais.
📊 Données financières Cours, bilans, news : appels directs aux APIs publiques (FMP, Polygon, Yahoo, FRED) avec les clés que tu fournis ou les endpoints gratuits.
💳 Paiement Lemonsqueezy gère le checkout. Ils nous envoient juste un webhook avec ton email pour activer la licence. Aucune autre donnée transite par nos serveurs.
📈 Analytics Plausible.io en mode anonyme (cookieless, IP hashée). Tu vois nos stats publiques. Aucun tracking individuel possible.

Ce qu'on ne peut PAS faire (même si on voulait)

  • Lire ton patrimoine ou tes positions — il n'a jamais quitté ton navigateur
  • Lire tes prompts d'analyse — ils vont directement au LLM provider
  • Lire tes clés API — elles sont chiffrées avec ta passphrase
  • Vendre ton activité à des annonceurs — on n'a pas la donnée
  • Te subpoena (même sous demande légale) tes analyses — on ne les a jamais eues

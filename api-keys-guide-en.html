🔑 How to get an AI API key & financial data

Practical 2026 guide — all keys supported by Alpha (14 LLMs + 11 data providers). Direct links, steps, free tier flagged.

📌 TL;DR — 7 free AI providers let you use Alpha without spending a cent (GitHub Models, Cerebras, Mistral, NVIDIA, Cloudflare, HuggingFace, Cohere). One key is enough to start. For pro analyses (10-K, DCF, complex tax), upgrade to Claude / GPT-4o / Gemini Pro (~$0.005-0.10 per analysis).

🚀 Get started in 30 seconds

  1. Pick a free provider below (recommended: GitHub Models if you already have a GitHub account).
  2. Click the direct link, create an account or generate a token.
  3. Copy the key.
  4. Open Alpha → Settings → API Keys → paste the key. Encrypted locally (AES-GCM 256).
  5. Run your first analysis 🎉

🎁 Free AI providers (7)

These providers offer a free tier sufficient for casual to intensive usage. Ideal to start without commitment.

🐙 GitHub Models 100% free

Easiest if you already have a GitHub account. Reasonable rate-limit. Models available: GPT-4o, Phi-4, Llama 3, DeepSeek, Mistral.

https://github.com/settings/tokens
  1. Sign in to your GitHub account.
  2. Go to Settings → Developer settings → Personal access tokens → Fine-grained tokens (or Classic).
  3. Click Generate new token. Give it a name (e.g., "Alpha Terminal").
  4. No special scope needed for GitHub Models. Create it and copy the token (starts with ghp_ or github_pat_).

⚡ Cerebras free tier

Ultra-fast inference (Llama 3.3 70B at >2000 tok/s). Generous free tier: 1M tokens/day.

https://cloud.cerebras.ai/?tab=api-keys
  1. Go to cloud.cerebras.ai, sign up with your email.
  2. API Keys tab, click Create API Key.
  3. Copy the key (starts with csk-).

🇫🇷 Mistral AI experiment tier

European models. Free experimentation tier for Mistral Small/Medium. Codestral (code) included.

https://console.mistral.ai/api-keys/
  1. Go to console.mistral.ai, sign up.
  2. Activate the Experiment plan (free, soft rate-limit).
  3. API Keys → Create new key.

🟢 NVIDIA NIM 1,000 free credits

Catalog of open-source models hosted by NVIDIA (Llama, Nemotron, DeepSeek). 1,000 free credits at signup.

https://build.nvidia.com/explore/discover
  1. Sign up at build.nvidia.com with your email.
  2. Pick any model, click Get API Key.
  3. Copy the key (starts with nvapi-).

☁️ Cloudflare Workers AI 100K req/day

Open-source models hosted on Cloudflare's edge. 100,000 free requests/day.

https://dash.cloudflare.com/profile/api-tokens
  1. Create a Cloudflare account (free).
  2. Note your Account ID from the dashboard (right sidebar).
  3. Go to My Profile → API Tokens → Create Token. Choose the Workers AI template.
  4. Alpha asks for both the token AND the Account ID.

  1. Sign up at huggingface.co.

  3. Copy the token (starts with hf_).

🟦 Cohere trial key

Good at classification, embeddings, and multilingual RAG. Free trial key at signup.

https://dashboard.cohere.com/api-keys

🔒 Key security in Alpha

  • Local AES-GCM 256-bit encryption with PBKDF2 derivation (100,000 iterations) from a master password you choose.
  • No Alpha server — the app calls provider APIs directly from your browser. Verifiable via DevTools → Network.
  • Local IndexedDB storage — your keys never leave your device.
  • Encrypted exportable backup (.atb) — for migration or backup.

❓ FAQ

I have only one key — do all modules work?

Yes. Alpha's smart router automatically picks the best available provider per module. With a single key, all 46 modules run. The more you add, the smarter the routing (e.g., Grok for X sentiment, Claude for 10-Ks, Gemini for long PDFs).

How much does it cost per month?

With free providers only: €0. With a free + Claude/GPT mix for critical analyses: $2-15/month. Power user (3+ analyses/day, 10-K, research-agent): $15-50/month. Details on the API costs page.

What if I forget my master password?

No recovery possible (the price of 100% local security). You can reset the vault → you lose stored keys and need to re-paste them. Tip: export an encrypted backup regularly and keep the password safe (password manager).

Are API keys transmitted to Alpha?

No. Never. The app runs client-side: your keys live only in your browser's IndexedDB, and AI requests go directly from your browser to the provider. You can put the app offline (airplane mode) after loading and analyses will keep going normally.

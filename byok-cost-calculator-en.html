🛡️ How to cap your budget

Three complementary mechanisms guarantee you'll never pay more than expected:

1. Monthly cap inside Alpha — Settings → Advanced → "Monthly budget cap". Set a value ($ or €, e.g. $5/month). Alpha tracks your usage in real time and blocks new calls once the cap is hit. Visual notification in the top bar.

2. Per-analysis cap — Settings → Advanced → "Per-analysis budget cap". Set a threshold ($) above which the analysis is canceled before sending. Safety net against heavy analyses (10-K on Berkshire = ~30k input tokens).

3. Provider-side limits — each provider (Anthropic, OpenAI, Google) lets you set hard limits in your account. E.g. Anthropic Console → Settings → Spend Limits. Set a hard cap (e.g. $20/month). Beyond that, the provider itself blocks requests.

With these 3 cumulative safeguards, surprise risk is near zero. Recommendation: start with an Alpha cap at $5/month, observe your real usage for 2-3 weeks, adjust.

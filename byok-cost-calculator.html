💡 Pourquoi BYOK = moins cher que les SaaS classiques

Les SaaS d'analyse IA (Simply Wall St, Koyfin AI, Quartr Pro...) facturent 5 à 10× le coût brut de l'API qu'ils utilisent en backend. Pourquoi ? Parce qu'ils doivent absorber :

  • Leur infrastructure cloud (proxies, load balancers, monitoring) : 30-40% du prix
  • Leur support client et leur équipe ops : 20-30%
  • Leur marge commerciale (ils sont une boîte, pas une asso) : 30-50%
  • Le risque de churn (ils paient même si tu n'utilises pas)

Avec BYOK, tu paies l'IA directement chez Anthropic / OpenAI / Google, au prix exact de leur facturation. Alpha n'est qu'une couche d'orchestration de prompts (9,99€/mois). Aucune marge sur l'IA.

Comparaison concrète pour 50 analyses/mois :

  • SaaS équivalent : ~30-50€/mois (forfait incluant l'IA)
  • Alpha BYOK : ~15€/mois (9,99€ Alpha + ~5€ API au prix coûtant)
  • Alpha BYOK avec free tier : 9,99€/mois (GitHub Models / Cerebras gratuits)

Économie typique : 50 à 80% sur 12 mois. Et bonus : si tu n'utilises pas l'app pendant 1 mois, tu ne paies que l'abonnement Alpha — l'API à $0.

🛡️ Comment plafonner ton budget

Trois mécanismes complémentaires garantissent que tu ne paieras jamais plus que prévu :

1. Plafond mensuel dans Alpha — Settings → Avancé → "Budget cap mensuel". Tu fixes une valeur ($ ou €, ex: 5€/mois). Alpha calcule en temps réel ta consommation et bloque les nouveaux appels une fois le cap atteint. Notification visuelle dans la barre supérieure.

2. Plafond par analyse — Settings → Avancé → "Budget cap par analyse". Tu fixes un seuil ($) au-dessus duquel l'analyse est annulée avant l'envoi. Garde-fou contre les analyses lourdes (10-K sur Berkshire = ~30k tokens d'input).

3. Limites côté provider — chaque provider (Anthropic, OpenAI, Google) permet de fixer des hard limits dans ton compte. Ex: Anthropic Console → Settings → Spend Limits. Tu peux définir un cap dur (ex: $20/mois). Au-delà, le provider lui-même bloque les requêtes.

Avec ces 3 garde-fous cumulables, le risque de mauvaise surprise est quasi-nul. Recommandation : commence avec un cap Alpha à 5€/mois, observe ta vraie consommation pendant 2-3 semaines, ajuste.

❓ FAQ

Qu'est-ce que le BYOK ?

BYOK = Bring Your Own Key. Tu apportes ta propre clé API (Claude, OpenAI, Gemini, Mistral...) et tu paies l'IA directement chez le provider, au prix coûtant. Alpha n'ajoute aucune marge sur l'IA.

Pourquoi le BYOK est moins cher qu'un SaaS classique ?

Les SaaS d'analyse IA prennent 5 à 10× le coût brut de l'API pour couvrir l'infra, le support et leur marge. Avec BYOK tu paies seulement l'abonnement Alpha (9,99€/mois) + l'IA au prix réel. Gain typique : 50 à 80% vs SaaS équivalent.

Puis-je utiliser une clé API gratuite ?

Oui. GitHub Models, Cerebras et Mistral offrent des tiers gratuits (rate-limités) qui couvrent l'usage casual. Avec ces clés, ton coût total = 9,99€/mois (abonnement Alpha uniquement).

Comment éviter les mauvaises surprises de facture API ?

Dans Settings → Avancé, tu peux fixer un plafond mensuel ($ ou €) sur ta consommation IA. Une fois atteint, Alpha bloque les nouveaux appels. Les providers (Anthropic, OpenAI, Google) permettent aussi de fixer des limites côté compte.

Mes clés API sont-elles en sécurité ?

Oui. Chiffrement AES-GCM 256 + PBKDF2 100K iterations, en local sur ton appareil. Le mot de passe maître ne quitte jamais ta machine. Aucun serveur Alpha ne voit tes clés.
